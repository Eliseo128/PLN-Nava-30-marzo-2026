### Ejemplo original

**Proyecto:** Asistente del Teorema de Pitágoras

**Descripción:**  
Este programa simula un agente de IA básico que ayuda al usuario a:  
- Calcular lados de un triángulo rectángulo  
- Explicar el teorema de Pitágoras  
- Guardar historial de consultas  

**Lenguaje:** Python  
**Nivel:** Principiante  
**Tecnologías:**  
- SQLite (almacenamiento de memoria)  
- Programación estructurada (if, while, funciones)

### Ejemplo 1

**Proyecto:** Conversor de Unidades Inteligente

**Descripción:**  
Este programa actúa como un asistente de conversión que permite al usuario:  
- Convertir entre diferentes unidades (longitud, masa, volumen)  
- Mostrar la fórmula utilizada en cada conversión  
- Guardar un historial de conversiones realizadas  

**Lenguaje:** Python  
**Nivel:** Principiante  
**Tecnologías:**  
- SQLite (registro de conversiones)  
- Programación estructurada (funciones, menú interactivo)

### Ejemplo 2

**Proyecto:** Registro de Hábitos Diarios

**Descripción:**

### Ejemplo 3

**Proyecto:** Mini Calculadora de IMC con Historial

**Descripción:**  
Este programa simula un agente de salud básico que ayuda al usuario a:  
- Calcular el Índice de Masa Corporal (IMC)  
- Interpretar el resultado (bajo peso, normal, sobrepeso, etc.)  
- Guardar un historial de cálculos con fecha y resultado  

**Lenguaje:** Python  
**Nivel:** Principiante  
**Tecnologías:**  
- SQLite (historial de mediciones)  
- Programación estructurada (entrada de usuario, validaciones, funciones)
